# wheel wow!!!

平时工作 or 无聊写的小轮子


#### copyText 复制黏贴板插件
- 支持h5，PC
- `success`, `error`回调
- 判断浏览器是否支持该事件接口

#### heatmap 热力图
- 老大提的小需求 一两个小时完成
- 点击页面会间隔(10s)存到`sessionLocation`。 刷新后会把存在`sessionLocation` 绘制成热力图展示在页面上

#### validator 小型表单验证插件
- 移动端简单的验证插件
- 支持自己配置验证项 支持多重验证  



#### cookie 封装了一些常用的api

- get, set, remove, clean, getAll
